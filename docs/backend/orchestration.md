# Orchestration

## Overview
Backend orchestration in LEAF is encoded directly in graph topology: branch, join, iterate, and schedule behaviors are all visible in wiring.

## When to use
Use this page when coordinating multiple backend operations with explicit control flow.

## Example
Combine `gate`, `mix`, and `loopyspell` for conditional and iterative orchestration while keeping payload transforms in LEAFlisp.
