# Performance Issues

## Overview
Performance problems in LEAF are often structural: oversized monolithic graphs, overuse of asynchronous coordination, or unnecessary repeated transforms.

## When to use
Use this page when latency or throughput regresses.

## Example
Refactor repeated subgraphs into reusable abstractions and observe behavior with targeted monitoring branches.
