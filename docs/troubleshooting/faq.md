# FAQ

## Overview
This FAQ captures recurring practical questions for LEAF users.

## When to use
Use this page for quick answers before deep troubleshooting.

## Example
Q: Why is my node not executing?
A: Check whether required inputs arrived, whether the branch is anchored, and whether synchronous fan-in is waiting.

Q: When should I use `memoryio`?
A: Only for temporary runtime state that does not need durability.
