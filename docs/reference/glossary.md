# Glossary

## Overview
Canonical LEAF terms from the source corpus.

## When to use
Use this page for quick terminology alignment during design and review.

## Example
Key terms:
- Dataflow edge: left-to-right data propagation edge. See [Dataflow Edge](../core-concepts/edge-types/dataflow.md).
- Lambda edge: top-to-bottom context edge. See [Lambda Edge](../core-concepts/edge-types/lambda.md).
- Anchor edge: control edge that can inactivate execution branches. See [Anchor Edge](../core-concepts/edge-types/anchor.md).
- LEAFlisp: Lisp dialect used for text-domain data transformation. See [Leaflisp Node](../core-concepts/node-types/leaflisp.md).
- Chronos: utility node enabling asynchronous handling of multiple incoming lines. See [Chronos Node](../core-concepts/node-types/chronos.md).
- MemoryIO: temporary runtime storage node. See [MemoryIO Node](../core-concepts/node-types/memoryio.md).
