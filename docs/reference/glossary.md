# Glossary

## Overview
Canonical LEAF terms from the source corpus.

## When to use
Use this page for quick terminology alignment during design and review.

## Example
Key terms:
- Dataflow edge: left-to-right data propagation edge.
- Lambda edge: top-to-bottom context edge.
- Anchor edge: control edge that can inactivate execution branches.
- LEAFlisp: Lisp dialect used for text-domain data transformation.
- Chronos: utility node enabling asynchronous handling of multiple incoming lines.
- MemoryIO: temporary runtime storage node.
