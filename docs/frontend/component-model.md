# Component Model

## Overview
In LEAF, components are node-level units. UI composition happens by wiring element nodes with transformation and control nodes.

## When to use
Use this page when breaking down larger interfaces into reusable graph fragments.

## Example
Treat each interaction region (form, display, media control) as a subgraph with clear input/output boundaries.
