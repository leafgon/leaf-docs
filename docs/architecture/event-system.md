# Event System

## Overview
LEAF workflows react to arriving data and interaction events. Inports, UI element outputs, and external I/O integrations all act as event sources that inject values into the dataflow plane.

## When to use
Use this page when modeling reactive behavior and event-driven orchestration.

## Example
A UI event emitted by an element node can trigger a transformation chain and then persist or render the result.
