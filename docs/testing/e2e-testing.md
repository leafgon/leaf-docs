# E2E Testing

## Overview
E2E testing validates full workflow behavior under representative scenarios including event ordering and integration responses.

## When to use
Use this page for release confidence on user-facing or business-critical LEAF workflows.

## Example
Run end-to-end scenarios that include realistic inputs, async conditions, and expected final outputs in `screenio`/`outport`.
