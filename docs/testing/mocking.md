# Mocking

## Overview
Mocking in LEAF testing isolates graph logic from unstable external integrations.

## When to use
Use this page when tests depend on network APIs or nondeterministic sources.

## Example
Replace external API nodes with deterministic payload producers while validating downstream transforms and branching.
