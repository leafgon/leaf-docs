# Unit Testing

## Overview
Unit testing in LEAF is primarily expression-level and node-local behavior validation.

## When to use
Use this page when verifying LEAFlisp logic such as arithmetic, branching, list/dictionary operations, and schema mapping.

## Example
Validate critical transforms like `parse`, `get`, `map`, and `reduce` with edge-case inputs.
