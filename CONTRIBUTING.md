# Contributing

## Overview
This docs set is meant to evolve with the platform. Keep edits scoped, source-backed, and cross-linked.

## When to use
Use this guide when proposing changes to LEAF documentation in this repository.

## Example
Recommended workflow:
1. Pick one page or one tightly scoped topic.
2. Update content using agreed LEAF documentation conventions (or clearly label assumptions).
3. Add or update cross-links.
4. Keep Mermaid diagrams simple and renderable.
